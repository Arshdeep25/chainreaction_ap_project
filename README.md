# Chain Reaction Game
## The game
It is an implementation of the chainreaction game in JAVA using javafx.
## Implementation
Javafx was used as the GUI library.
## Compilation
Compile and run the java code using :<br>
<code>javac main_files/*.java</code><br>
<code>java main_files.MainPage</code>

It is was originally made as a project for course Advance Programming.
